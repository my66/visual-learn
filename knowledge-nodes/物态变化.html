能量升高 = 吸热

“往上爬楼梯，累了要‘吸’气”

能量降低 = 放热

“下楼梯很轻松，把热量‘放’出”

吸热 (能量升高)

熔化 (Melting)

→

固态 → 液态

(能量：低 → 中)

举例: 冰化成水；碘在加热超过熔点时可由固→液（碘也可在较低压强/通风条件下直接升华）。

汽化 (Vaporization)

→

液态 → 气态

(能量：中 → 高)

举例: 水烧开（沸腾）；湿衣服变干（蒸发）。

升华 (Sublimation)

→

固态 → 气态（跳级）

(能量：低 → 高)

口诀: “‘升’就是往上跳，从1楼跳到3楼。”

举例: 干冰；樟脑球；碘；长期不开门的冰箱里，冰块会变少（升华）。

放热 (能量降低)

凝固 (Freezing)

→

液态 → 固态

(能量：中 → 低)

举例: 水结冰。

液化/凝结 (Condensation)

→

气态 → 液态

(能量：高 → 中)

举例: 玻璃上“结露”；“白气”本质是小水滴（气→液），如冰棒冒白雾、冬天口中哈气；云滴/雨的形成（先凝结成微小水滴，聚并下落成雨）。

凝华 (Deposition)

→

气态 → 固态（跳级）

(能量：高 → 低)

口诀: “‘凝’就是凝结往下掉，从3楼掉到1楼。”

举例: 结霜；雪花的形成（在高空中水汽直接在凝结核上凝华为冰晶，聚合成雪花）。

相变时温度恒定（潜热）

纯物质在标准压强下发生相变（如熔化、沸腾、凝固、凝华等）时，温度保持不变，吸收/放出的热量用于改变物态而非升/降温。 例如：0 °C 的冰吸热变成 0 °C 的水；100 °C 的水沸腾时温度不再上升。

蒸发 vs 沸腾（都属汽化）

  • 蒸发：表面进行、任何温度、速度慢。
  • 沸腾：达到沸点后，液体内部与表面同时汽化，产生气泡，速度快。
  • 提高温度、增大风速、增大表面积、降低湿度 → 蒸发更快。

易错判断（考点强化）

  • “白气”≠水蒸气，白气是小水滴（气→液，液化/凝结）。
  • 雪/霜：气→固（凝华），不是先变成液体。
  • 碘可升华也可熔化，取决于温度与压强/环境条件。
  • 相变时温度不变；非相变的升温/降温时温度才改变。
